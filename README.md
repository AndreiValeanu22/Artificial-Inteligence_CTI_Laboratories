# Laboratoare Inteligență Artificială (CTI)

Acest depozit conține **materialele de laborator** pentru disciplina de **Inteligență Artificială**, organizate pe foldere `LAB1` … `LAB11`. Fiecare folder este **autonom** (în general un notebook principal și, uneori, fișiere auxiliare).

## Conținut pe laboratoare

| Folder | Temă principală |
|--------|------------------|
| **LAB1** | Introducere Python |
| **LAB2** | Căutare: algoritm **A\*** |
| **LAB3** | Căutare locală: **Hill Climbing** |
| **LAB4** | **PCSP** (probleme cu restricții) |
| **LAB5** | **MCTS** (Monte Carlo Tree Search) |
| **LAB6** | Învățare automată: **Random Forest** |
| **LAB7** | Regresie **liniară** |
| **LAB8** | Regresie **logistică** |
| **LAB9** | Rețele neuronale: **introducere MLP** |
| **LAB10** | Modele probabilistice: **seminar Bayes** |
| **LAB11** | **Eliminarea variabilelor** în rețele Bayesiene (inferență exactă pe factori) |

## Cum lucrezi local

1. Instalează **Python 3** (versiunea folosită la curs, de preferință 3.10+).
2. Deschide notebook-ul din folderul dorit în **Jupyter Notebook**, **JupyterLab** sau **VS Code**.
3. Rulează celulele **de sus în jos** la prima parcurgere, ca să ai funcțiile și exemplele definite în ordine.

## Note

- Fișierele din `.ipynb_checkpoints` nu sunt necesare pentru lucru și **nu** ar trebui versionate; dacă apar local, le poți ignora sau șterge.
- Pentru **LAB11**, după rezolvarea cerințelor, poți folosi și secțiunea de la finalul notebook-ului intitulată aproximativ **„Ghid de lectură”**, cu explicații suplimentare și legături spre resurse.
